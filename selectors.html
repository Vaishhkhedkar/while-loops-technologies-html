welcome to CSS

  1. css stands for cascading style sheet
  2. There are three types of CSS: external, internal, and inline.
  3. External CSS commands are written in a separate file from the main HTML page
  4. These are then linked to each web page by a command in the HTML file.
  5. The display property is used to specify how an element is shown on a web page.
